통계 검정 Frame Work End to End¶

통계지식 도대체 어디에 어떻게 쓰는데?¶

NOTE: Post - hoc

One-way ANOVA를 하면 3 개 중에 적어도 하나는 통계적으로 다른지를 판단할 수 있음. 그러나, 어떤 쌍 (pair)이 서로 다른지는 알수 없다.

ex)

A, B, C 세 그룹 중 하나라도 다르다는 One-way Anova 분석 결과가 나왔다면

A vs. B

B vs. C

A vs. C

를 비교하고자 할 때, 다시 t-test를 각각 3번씩 할 수도 있으나, post-hoc test를 하면, 한번에 세가지 답을 구할 수 있음.

    ppp = len(list(itertools.combinations(unique_levels, 2)))

    unique_combi2 = []

    for i in range(n_levels-1):
        for j in range((i+1), n_levels):
            unique_combi.append(unique_levels[i] + " vs " + unique_levels[j])
            unique_combi2.append([i, j])

    ### more than 3 groups
    if n_levels > 2:
        ##normality test
        warnings.filterwarnings('ignore')

        #df = pd.DataFrame(data, columns=['value', 'treatment'])

        # the "C" indicates categorical data
        model = ols(temp_formula, temp_data).fit()
        predictValues = model.predict()
        residuals = temp_data.loc[:, 'avg_start_bat11_bat_soc'] - predictValues  # 잔차
        nor_pvals = stats.shapiro(residuals)[1]

        if nor_pvals is None:
            main.iloc[k, 2] = "Kruskal-Wallis test"
            #kruskal test
            y = temp_data.loc[:, 'avg_start_bat11_bat_soc']
            x = temp_data.loc[:, varlist[k]]

            y = np.array(y)
            label, idx = np.unique(x, return_inverse=True)
            groups = [y[idx == i] for i, l in enumerate(label)]
            H, p = stats.kruskal(*groups)

            main.iloc[k, 3] = p

            ##post-hoc test
            if main.iloc[k, 3] < 0.05:
                post_hoc_tb = pd.DataFrame(columns=["pair", "Diff", "Lower CI", "Upper CI"])
                post_hoc_tb = post_hoc_tb.assign(pair=unique_combi)
                post_hoc_tb = post_hoc_tb.fillna(0)

                ##sidak correction (#wilcox ranksum)
                for q in range(0, ppp):
                    s, mp = stats.mannwhitneyu(groups[unique_combi2[q][0]], groups[unique_combi2[q][1]], alternative='two-sided')

                    ct1 = len(groups[unique_combi2[q][0]])  #items in dataset 1
                    ct2 = len(groups[unique_combi2[q][1]])  #items in dataset 2
                    alpha = 0.05       #95% confidence interval
                    N = stats.norm.ppf(1 - alpha/2) # percent point function - inverse of cdf

                    # The confidence interval for the difference between the two population
                    # medians is derived through these nxm differences.
                    diffs = sorted([i-j for i in groups[unique_combi2[q][0]] for j in groups[unique_combi2[q][1]]])

                    # For an approximate 100(1-a)% confidence interval first calculate K:
                    kv = int(round(ct1*ct2/2 - (N * (ct1*ct2*(ct1+ct2+1)/12)**0.5)))

                    # The Kth smallest to the Kth largest of the n x m differences
                    # ct1 and ct2 should be > ~20
                    CI = (diffs[kv], diffs[len(diffs)-kv])

                    post_hoc_tb.iloc[q, 1] = round(np.median(diffs), 5)
                    post_hoc_tb.iloc[q, 2] = round(CI[0], 5)
                    post_hoc_tb.iloc[q, 3] = round(CI[1], 5)

                post_hoc.append(post_hoc_tb)
