Linear Regression - 초식을 깊숙히 익히자¶

Linear Regression은 맞추고자 하는 y가 아래와 같은 1차 관측치 들의 합이라는 가정에서 출발한다¶

\[\Large\hat{y} = w_0 + w_1x_1 + w_2x_2 + ...\]
  1. Feature : \(x_1, x_2,...\) (주어진 데이터)
  2. Weight : \(w_2, w_2,...\) (구해 내야하는 값, Fix but Unkown)
  3. Bias(or intercept) : \(w_0\) (구해 내야하는 값, Fix but Unkown)
  4. Predicted Value : \(\hat{y}\) (Feature가 주어졌을 때 y와 유사하면 좋은 값)

그렇다면 주어진 데이터(Feature) 를 기반으로 \(\hat{y}\) 와 \(y\) 의 차이를 최소화 하는 Weight를 구하면 되네?¶

두번째 시도 : Gradient Descent Method¶

  • Gradient의 기하학적 의미
    1. Gradient 와 같은 방향은 함수가 가장 빠르게 증가하는 방향
    2. Gradient 와 반대 방향은 함수가 가장 빠르게 감소하는 방향
    3. 1차원 에서는 미분값 = 기울기

With initial \(x^{(0)}\), calculate the following equation :

\[\begin{equation} x^{(k+1)} = x^{(k)} - \alpha \nabla f(x^{(k)}) \end{equation}\]
  1. 현재위치에서 : \(x^{(k)}\)
  2. 함수(Loss Function)값이 작아지는 방향을 찾아서 : \(- \alpha \nabla f(x^{(k)}\) (\(\alpha\)는 Tuning Factor)
  3. 이동하는 : \(x^{(k+1)}\)
  4. 알고리즘

# 함수로 만들어 보기
def steepest_descent_2d(func, gradx, grady, x0, MaxIter=10, learning_rate=0.25, verbose=True):
    paths = [x0]
    fval_paths = [func(x0[0], x0[1])]
    if verbose:
        print("{:<4} {:<25} {:<40}".format("i", "x", "f(x)"))
    for i in range(MaxIter):
        grad = np.array([gradx(*x0), grady(*x0)])
        x1 = x0 - learning_rate * grad
        fval = func(*x1)
        if verbose:
            print("{:<3}: {:<25} {:<40}".format(i, str(x1), fval))
        x0 = x1
        paths.append(x0)
        fval_paths.append(fval)
    paths = np.array(paths)
    paths = np.array(np.matrix(paths).T)
    fval_paths = np.array(fval_paths)
    return(x0, fval, paths, fval_paths)

Descent 계열 알고리즘의 핵심 사항¶

  1. Weight의 초기값을 무엇으로 할까?
  2. Search 방향을 어떻게 정할까?
  3. Learning Rate를 어떤 값으로할까?

Descent 계열이라고? 그럼 Descent 계열에 Gradient Descent 말고 또 있니? 많다!¶

  1. Weight의 초기값을 무엇으로 할까?
  2. Search 방향을 어떻게 정할까? \(\to - \alpha \nabla^2 f(x^{(k)})^{-1}\nabla f(x^{(k)})\)
  3. Learning Rate를 어떤 값으로할까?
\[\begin{equation} x^{(k+1)} = x^{(k)} - \alpha \nabla^2 f(x^{(k)})^{-1}\nabla f(x^{(k)}) \end{equation}\]

# 함수로 만들어보기
def newton_descent_2d(func, gradx, grady, hessian, x0, MaxIter=10, learning_rate=1, verbose=True):
    paths = [x0]
    fval_paths = [func(x0[0], x0[1])]
    if verbose:
        print("{:<4} {:<25} {:<40}".format("i", "x", "f(x)"))
    for i in range(MaxIter):
        grad = np.array([gradx(*x0), grady(*x0)])
        hess = hessian(*x0)
        x1 = x0 - learning_rate * np.linalg.solve(hess, grad)
        fval = func(*x0)
        if verbose:
            print("{:<3} {:<25} {:<40}".format(i, str(x0), fval))
        x0 = x1
        paths.append(x0)
        fval_paths.append(fval)
    paths = np.array(paths)
    paths = np.array(np.matrix(paths).T)
    fval_paths = np.array(fval_paths)
    return(x0, fval, paths, fval_paths)

1. scipy.optimize 사용하기¶

\[\begin{equation} loss(w) = \frac{1}{N}\sum_{i=1}^N |w_0 x_i^2 + w_1x_i + w_2 - y_i|^2 \end{equation}\]

2. Steepest Descent 사용하기¶

\[\begin{equation} loss(w) = \frac{1}{N}\sum_{i=1}^N |w_0 x_i^2 + w_1x_i + w_2 - y_i|^2 \end{equation}\]
\[\begin{split}\begin{equation} \nabla loss(w) = \frac{2}{N}\sum_{i=1}^N (w_0 x_i^2 + w_1x_i + w_2 - y_i) \begin{bmatrix} x_i^2\\ x_i\\ 1 \end{bmatrix} \end{equation}\end{split}\]

3. Newton method 사용하기¶

\[\begin{equation} loss(w) = \frac{1}{N}\sum_{i=1}^N |w_0 x_i^2 + w_1x_i + w_2 - y_i|^2 \end{equation}\]
\[\begin{split}\begin{equation} \nabla loss(w) = \frac{2}{N}\sum_{i=1}^N (w_0 x_i^2 + w_1x_i + w_2 - y_i) \begin{bmatrix} x_i^2\\ x_i\\ 1 \end{bmatrix} \end{equation}\end{split}\]
\[\begin{split}\begin{equation} \nabla^2 loss(w) = \frac{2}{N}\sum_{i=1}^N \begin{bmatrix} x_i^4 & x_i^3 & x_i^2\\ x_i^3 & x_i^2 & x_i\\ x_i^2 & x_i & 1 \end{bmatrix} \end{equation}\end{split}\]
